Enhancement Two - Algorithms Narrative

The artifact selected for this enhancement is a Python-based inventory management program that was originally developed earlier in the Computer Science program. The application allows users to add, view, sort, and search for inventory items using a command-line interface. For this milestone, the artifact was enhanced to focus specifically on algorithms and data structures by improving how data is organized, accessed, and processed. These enhancements included implementing a binary search algorithm and maintaining a consistently sorted data structure to improve efficiency and performance.
This artifact was selected because it provides a clear and practical way to demonstrate the application of algorithmic principles within a real-world scenario. Originally, the program relied on a linear search method, which becomes inefficient as the size of the dataset increases. By implementing binary search, the time complexity was improved from O(n) to O(log n), demonstrating a stronger understanding of algorithm efficiency and scalability. Additionally, the inventory is now automatically sorted after each insertion, ensuring that the data structure supports the binary search algorithm. These improvements showcase skills in algorithm design, data organization, and performance optimization, which are critical components of computer science practice.
The enhancements made in this milestone align with the course outcomes related to designing and evaluating computing solutions using algorithmic principles and applying appropriate data structures. The improvements demonstrate the ability to analyze trade-offs between different approaches, such as choosing binary search over linear search for improved efficiency. While significant progress has been made toward these outcomes, additional refinement and expansion—such as implementing more advanced data structures or persistent storage—may further strengthen coverage of these competencies in future enhancements.
Through the process of enhancing this artifact, several important concepts were reinforced. One key learning experience was understanding the importance of maintaining sorted data to support more efficient algorithms such as binary search. This required not only implementing the search algorithm but also restructuring how data is managed within the program. A challenge encountered during this process was ensuring that the inventory remained consistently sorted after each modification while preserving the program’s original functionality. Overcoming this required careful testing and validation of logic. Overall, this enhancement deepened my understanding of algorithm efficiency, data structures, and how design decisions directly impact performance in software applications.
Research into computer science careers has also influenced how this artifact was developed and enhanced. According to ComputerScience.org, careers in computer science require strong problem-solving abilities and a solid understanding of algorithms and data structures, as these are fundamental to building efficient and scalable systems. Additionally, the job search process emphasizes the importance of showcasing practical projects and technical competencies through portfolios and interviews. This reinforced the importance of not only building functional applications but also demonstrating measurable improvements and technical depth within those applications.
